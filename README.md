## Instructions (Operation Manual)

1. Open the project in a web browser (via GitHub Pages link or local index.html file).
2. Click the "START" button in the center of the screen.
3. The system will begin a 5-minute real-time playback of recorded sound events.
4. Each circle represents a sound event:
   - Position on the horizontal axis = time
   - Vertical level = perceived distance (outside, corridor, self, unknown)
5. The expanding ripple animation lasts 0.8 seconds when a sound appears.
6. The outer ring remains visible to represent the recorded event.
7. After 5 minutes, the visualisation ends.

---

## Project Description

This project documents five minutes of real sound events in my dormitory.

During the observation, I recorded a continuous five-minute video and manually logged each audible event with timestamps using my phone.

Rather than analysing sound properties such as frequency or volume, this project focuses on spatial perception — how sound is experienced in terms of distance.

The visualisation translates subjective auditory experience into structured data using p5.js.
